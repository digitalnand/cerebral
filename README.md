# cerebral
![mandelbrot by Erik Bosman](docs/mandelbrot.png)
# Requirements
- make
- a c++23 compiler
# Usage
build the project
```bash
make
```
run bin/crbl providing it with a valid brainfuck program
```bash
bin/crbl mandelbrot.bf
```
